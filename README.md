# Análisis de viajes en Taxi – SQL Case Study

## Contexto
Este proyecto analiza datos históricos de viajes en taxi con el objetivo de identificar patrones de demanda, comportamiento de empresas y factores externos que influyen en la duración de los viajes.

El análisis se realizó utilizando bases de datos relacionales y técnicas de estadística para validar hipótesis sobre el impacto del clima en los tiempos de viaje.

---

## Objetivo
Explorar y analizar datos de viajes para:
- Segmentar empresas y tipos de viaje.
- Evaluar factores que influyen en la duración.
- Validar hipótesis estadísticas relacionadas con condiciones externas.

---

## Tecnologías utilizadas
- SQL  
- PostgreSQL  
- Python  
- pandas  
- Estadística inferencial  

---

## Proceso de análisis

1. **Exploración de datos**
   - Revisión de tablas y relaciones.
   - Identificación de variables relevantes.

2. **Consultas SQL**
   - Segmentación de viajes por empresa.
   - Cálculo de métricas agregadas.
   - Filtros por condiciones climáticas.

3. **Análisis estadístico**
   - Pruebas de hipótesis.
   - Comparación de duración de viajes bajo diferentes condiciones.

4. **Visualización**
   - Gráficos para interpretar resultados.

---

## Resultados principales

- Se identificaron diferencias significativas en la duración de los viajes según condiciones climáticas.
- Algunas empresas concentran mayor volumen de viajes en horarios específicos.
- El clima influye de manera directa en los tiempos promedio de traslado.

---

## Conclusiones

El análisis demuestra que los factores externos, como el clima, tienen un impacto estadísticamente significativo en la duración de los viajes. Estos resultados pueden ser útiles para optimizar la planificación de flotas y mejorar la experiencia del usuario.
